## Online Multiplayer Vision

The long-term multiplayer goal is not only direct online play, but a competitive online system.

### Planned expectations

- online player vs player matches
- player rating / ELO system
- win/loss based rating updates
- match result tracking
- support for ranked play

### Possible matchmaking directions

- Find Match system for ranked play
- Private lobby / room code system for playing with friends
- Public lobbies are optional and may be evaluated later

### Notes

This system is expected to require a separate technical planning phase because online multiplayer introduces synchronization, session handling, and rating system complexity that goes beyond local match flow.
